```
Frontend: Next.js 13+ (App Router) + React 18 + TypeScript
UI: Tailwind CSS + shadcn/ui + Lucide React
Backend: Supabase (PostgreSQL + API REST)
Hosting: Vercel (déploiement automatique)
```
